sanitizeIPv6(address) → string

Converts an IPv6 address to the canonical form stored and used by MediaWiki. JavaScript translation of the IP::sanitizeIP() function from the IPUtils library. Addresses are verbose, uppercase, normalized, and expanded to 8 words.

validCIDR() → boolean

Check that an IP range is within the CIDR limits. Most likely to be useful in conjunction with wgRelevantUserName. CIDR limits are hardcoded as /16 for IPv4 and /32 for IPv6.

Returns

boolean 

  • True for valid ranges within the CIDR limits, otherwise false (ranges outside the limit, single IPs, non-IPs).
